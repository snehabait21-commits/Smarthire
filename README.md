# 🎯 SmartHire — Job Application Tracker

A full-stack web app built with **Python (Flask)** to help job seekers manage and track their job applications intelligently.

## ✨ Features
- 🔐 User authentication (Signup / Login / Logout)
- 📋 Track job applications (Company, Role, Status, Date)
- 📊 Dashboard with stats (Total / Interviews / Offers / Rejected)
- 🔍 Resume Keyword Matcher — paste a job description and see which keywords are missing from your resume
- ✏️ Full CRUD — Add, Edit, Delete applications
- 📱 Responsive design with Bootstrap 5

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python, Flask, Flask-SQLAlchemy |
| Database | SQLite |
| Frontend | HTML, CSS, Bootstrap 5, JavaScript |
| Auth | Flask sessions, Werkzeug password hashing |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/snehabait21-commits/Smarthire.git
cd Smarthire

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env

# Run the app
python app.py
```

Visit `http://127.0.0.1:5000`
